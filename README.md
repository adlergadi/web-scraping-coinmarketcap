# Miniprojeto - Web Scraping de Criptomoedas

## Descrição do Projeto

Script em Python para coleta de dados de criptomoedas utilizando técnicas de web scraping.
O programa acessa o site CoinMarketCap (https://coinmarketcap.com/) e extrai informações relevantes sobre as principais criptomoedas listadas na página inicial.

## Dados coletados:

- Nome da criptomoeda

- Código (símbolo)

- Preço atual

- Market Cap

- Volume de negociação

- Variação percentual em 1 hora

- Variação percentual em 24 horas

- Variação percentual em 7 dias

As informações são armazenadas em um dicionário Python, permitindo posterior manipulação, análise ou armazenamento.

## Ferramentas

- Python

- Requests

- BeautifulSoup (bs4)

- Regex (re)
